## Description



**Npress** is an open source CMS based on [Nest](https://github.com/nestjs/nest) framework for API and [Nextjs](https://nextjs.org/) for client.



## Installation



```bash

$ npm install

```



## Running the app



```bash

# lcoal development

$ npm run start:local


# production mode

$ npm run start:prod

```



## Test



```bash

# unit tests

$ npm run test



# e2e tests

$ npm run test:e2e



# test coverage

$ npm run test:cov

```
